- submit form
- validate youtube url
    - /videos
    - playlist
    - sigular videos

- start fetching of all videos based on link provided
    - validate video cache for previously existing data for the submited link
        - if there are none, generate a new cache key for this task
        and mark in progress as true
    - start retreival of video links from submitted youtube url
        - when there are enough links to fill 1 page of a requested batch_size, extract their video information
    - when all links have been retreived, mark this job as completed

- for when extracting video information
    - validate video info cache for previously existing videos tied via cache key
    - check which page based on how many are already currently rendered on the DOM
    - for every video link in a batch
        - retrieve and cache (only those that have been requested based on the number of pages of videos have been rendered to the DOM)
            - video id
            - video title
            - video thumbnail (highest available quality in jpg or png)
            - video embed link
        - when informatino is successfully retreived, render the elemnt to the page

- for when pressing the load more button
    - if video link extraction is still in progress
        - check if there are enough videos in the cache to generate a new pages, if not then deny
    - if video link extraction is done and there are still more video links
        - extract video information for the next batch
